  • Weka Examples

The Weka Examples collection is a comprehensive collection of examples for the different versions of Weka in the form of an ANT project. You can access these examples as follows:

Snapshots

Through snapshots or releases (they contain a separate ZIP file with the ANT project)

Subversion

Through subversion

  • stable-3.8 version (3.8.x):

    https://svn.cms.waikato.ac.nz/svn/weka/branches/stable-3-8/wekaexamples/

  • developer version (3.9.x):

    https://svn.cms.waikato.ac.nz/svn/weka/trunk/wekaexamples/
