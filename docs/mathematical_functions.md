Mathematical functions implemented on dataset instances, like tan, cos, exp, log, and so on can be achived using one of the following filters:

* [AddExpression](https://weka.sourceforge.io/doc.stable-3-8/weka/filters/unsupervised/attribute/AddExpression.html) (Stable version)
* [MathExpression](https://weka.sourceforge.io/doc.stable-3-8/weka/filters/unsupervised/attribute/MathExpression.html) (Stable version)
